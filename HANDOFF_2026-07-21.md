## 2. Traps that will cost you hours

- **`paper/refs.bib`, NOT `references.bib`.** A sibling `paper/references/` directory of
  cited-paper PDFs shadows that basename in BibTeX's lookup, so `\bibliography{references}`
  opens the *directory* and dies with "Access is denied". This was masked for hours by a
  stale `main.bbl`. Do not rename it back.
- **Ablation dirs are `<ts>_<model>_<paraphrase>`.** Parallel one-per-model passes used to
  collide on a second-resolution timestamp and silently overwrite each other (cost ~3,600
  API calls). Fixed in `run_ablations.py`; a non-empty target dir is now a hard error.
- **Verify a run's identity from artifact `_meta`, not logs.** Both bugs above exited 0
  with logs reporting success.
- **Do not glob all ablation dirs into `--ablation-dirs`.** Two ceilings for the same
  model+dataset+strategy+paraphrase is a deliberate hard error. A 2-instance smoke dir was
  deleted for exactly this reason.
- **Windows console is cp1252.** No en-dashes/arrows in `print()`; matplotlib labels need
  real `â€“` (U+2013), LaTeX `--` renders literally in figures.
- **No system LaTeX.** Tectonic was downloaded to the session scratchpad; re-fetch from
  the tectonic-typesetting GitHub releases if the scratchpad is gone.
- **Nova Pro throttles hard** (EU cross-region profile; ~900 throttles in one pass). Qwen
  and DeepSeek do not. Interleave models round-robin across a worker pool â€” batching by
  model saturates the pool on the slow one. `--max-retries 8` took retry exhaustion to 0.
